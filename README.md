It focuses on building structured web pages, styling layouts, and improving overall front-end fundamentals. The project reflects hands-on learning by implementing the topics covered throughout the course.

- CSS

## Purpose

- Apply CSS styling and layout concepts  
- Gain hands-on experience through course-based learning
