# Udemy_project
This project was developed as part of a Udemy course to practice and apply the course concepts using HTML and CSS. It focuses on building structured web pages, styling layouts, and improving overall front-end fundamentals. The project reflects hands-on learning by implementing the topics covered throughout the course.

## Technologies Used
- HTML
- CSS

## Purpose
- Practice HTML structure and semantic elements  
- Apply CSS styling and layout concepts  
- Gain hands-on experience through course-based learning
